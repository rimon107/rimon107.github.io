**Data Collection and Validation Portal** is an enterprise web application developed for large-scale financial data collection, validation, and reporting workflows.

The system supported CSV ingestion, schema validation, user management, reporting, and analytics dashboards for regulatory data operations.

### Highlights

- Engineered a web platform for CSV upload, validation, management, and reporting.
- Built role-based access control and user-management features.
- Developed reporting tools and data-visualization dashboards.
- Containerized the application using Docker.
- Used Django, React, Redux, Oracle, and Docker in enterprise data workflows.

### Keywords

Django, React, Redux, Oracle, Docker, ETL, enterprise data systems, regulatory analytics.
