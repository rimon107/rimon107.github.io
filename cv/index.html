Projects

  • CRiC

    Protocol-scoped counterfactual risk certification framework for interactive autonomous driving.

    • Developed a protocol-scoped certification framework that evaluates a frozen autonomous-driving decision protocol rather than only a predictor.
    • Designed calibrated upper-risk-bound selection and abstention logic for safety-aware candidate choice under uncertainty.
    • Built streaming-friendly evaluation artifacts and reliability analyses for large-scale scene/candidate rollouts.
  • BucketKD

    Safety-aware bucket-based knowledge distillation framework for compressing end-to-end autonomous-driving motion planners.

    • Developed bucket-based knowledge distillation for end-to-end motion-planner compression.
    • Designed time-to-collision-guided waypoint attention and adaptive bucket construction.
    • Evaluated compact student planners in CARLA high-complexity driving scenarios.
  • Autonomous R/C Cars Cybersecurity Summer Camp

    Hands-on autonomous driving and cybersecurity education program using Donkey Car and Raspberry Pi platforms.

    • Built an end-to-end self-driving pipeline including data collection, CNN training, and on-vehicle deployment.
    • Designed demonstrations of safety-critical attack surfaces in connected and autonomous systems.
    • Taught high-school students Python, machine learning, robotics, and cybersecurity concepts.
  • Data Collection and Validation Portal

    Enterprise web platform for regulatory data ingestion, validation, reporting, and analytics.

    • Engineered CSV upload, schema validation, role-based access, and reporting workflows.
    • Built dashboard and analytics features for regulatory data operations.
    • Used Django, React, Redux, Oracle, and Docker.
  • Bangla Automated Speech Recognition Engine

    Samsung Research Bangladesh-funded project on low-resource Bangla speech recognition, voice command recognition, and speech/text resource development.

    • Contributed to end-to-end Bangla voice command recognition using contextual rescoring.
    • Helped prepare Bangla speech corpus resources from publicly available audio and text.
    • Worked on grapheme-to-phoneme modeling for non-trivial Bangla transcription problems.
    • Contributed to publications at ICASSP, LREC, and NAACL.
